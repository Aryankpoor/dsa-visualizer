# 🚀 DSA Visualizer

An interactive **Data Structures & Algorithms Visualizer** built to help understand how algorithms work step-by-step through clean animations and intuitive UI.

---

## 🌟 Features

* 📊 Visualize popular **sorting algorithms**
* 🎮 Step-by-step **array representation**
* 🎯 Clean and responsive UI

---

## 🧠 Algorithms Implemented

### 🔷 Sorting Algorithms

* Bubble Sort
* Selection Sort
* Insertion Sort
* Merge Sort
* Quick Sort

### 🔷 Searching Algorithms

* Binary Search
* Linear Search

### 🔷 DP Algorithms

* Knapsack
* Longest Common Subsequence

### 🔷 Graph Algorithms

* DFS (Depth First Search)
* BFS (Breadth First Search)

## 🎮 How to Use

1. Select an algorithm from the sidebar
2. Generate a new array (or use existing)
3. Click **Start**
4. Watch the algorithm execute step-by-step

---

## 💡 Key Design Decisions

* 🔹 **Global CSS for buttons & layout** → consistency across pages
* 🔹 **Animation-based architecture** → algorithms emit steps instead of direct sorting
* 🔹 **Modular structure** → easy to add new algorithms
